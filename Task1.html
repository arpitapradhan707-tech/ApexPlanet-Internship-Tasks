Task 1: HTML, CSS and JavaScript

Basics of Web Development

This simple webpage demonstrates the foundation of front-end development: structured HTML content, clean CSS styling, and a small JavaScript interaction.

What This Page Uses

HTML Structure

Headings, paragraphs, navigation links, sections, articles, an image area, and a button are used to create a clear webpage layout.

CSS Styling

Colors, spacing, typography, shadows, responsive grids, and hover effects make the design feel modern and natural.

JavaScript Alert

The button at the top uses JavaScript to show an alert message when it is clicked, adding a basic interaction.

Learning by Building

Web development starts with small pages like this one. HTML gives the page meaning, CSS shapes the visual experience, and JavaScript makes the page respond to the user.

Once these basics feel comfortable, it becomes easier to create forms, interactive menus, dashboards, portfolios, and complete websites.
